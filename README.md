conkeror-theme-zenburn
======================
Zenburn theme for conkeror

## Load theme ##

just place the directory into your conkeror config directory
[~/.conkeror.mozdev.org] and load the theme in your conkerorrc
[~/.conkerorc] with:

`theme_load("conkeror-theme-zenburn");`
